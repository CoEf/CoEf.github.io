Godot에서 3D 페인팅 셰이더(UV 기반 blit)를 만들 때, UV 심의 위치가 런타임
아티팩트에 직접 영향을 줍니다. 블렌더에서 언랩할 때 다음 기준을 지키면
UV 공간에서의 보간 문제를 줄일 수 있습니다.

## 심 배치 기준

- 눈에 잘 안 띄는 모서리(겨드랑이, 안쪽 이음새)에 배치
- 곡률이 큰 부분(팔, 다리 관절)은 심을 피하고 평평한 면에 배치
- 텍스처 페인팅이 자주 일어나는 표면(얼굴, 몸통 앞면)은 UV 아일랜드를
  가능한 한 크게, 심 없이 이어붙이기

## Mark Seam 단축키

`Edit Mode` → 엣지 선택 → `Ctrl+E` → `Mark Seam`

`U` → `Unwrap`으로 언랩 후 UV 에디터에서 아일랜드가 예상대로 나뉘었는지
확인합니다.

## Godot 임포트 시 확인할 점

블렌더에서 만든 UV가 Godot으로 넘어올 때 좌표계(V축 방향)가 반전되는
경우가 있어, 실제 셰이더에서 UV를 샘플링하기 전에 테스트 그리드 텍스처로
방향을 먼저 확인하는 것을 추천합니다.
